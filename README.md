# bewegt · dein Training

Persönliche Trainings- und Disziplin-App (Web-App, single-file).

- **Heute**: orbitales Kontrollzentrum — Ziele als antippbare Bubbles, Tagesring, Karussell mit Details
- **Plan**: Wochenplan als Timeline, Übungen mit Gewicht & Wiederholungen pro Krafteinheit
- **Fortschritt**: strenger Gesamt-Streak, Ampel-Kalender mit Nachtragen, Meilensteine, Wochenrückblick

Daten werden lokal im Browser gespeichert (localStorage); Backup über Export/Import im Plan-Tab.
